Copy2Paste Help

Drag a box over any part of a page and it's copied to your clipboard — either as plain text (default, and it now also reads text baked into images inside your selection) or as an image, whichever mode you pick.

How to use it

  1. Click the toolbar icon and choose Start selection (text) or Start selection (image) — or press a keyboard shortcut if you've set one (see below), which skips the menu entirely.
  2. Your cursor becomes a crosshair. Click and drag a box over the area you want.
  3. Release the mouse — it's copied. Paste it anywhere with Ctrl+V / Cmd+V (as text in text mode, as an image in image mode).
  4. Press Esc any time before releasing to cancel.

Reading text inside images (OCR)

If your text-mode selection includes a photo, screenshot, banner, video, or anything else with baked-in text, Copy2Paste automatically runs it through a local OCR engine and appends whatever it reads to the copied text. This only kicks in when an image or video is actually inside the selection, so ordinary text selections stay instant. When it does run, a small note ("Reading text in image…") appears for a few seconds while it works. Everything happens on your device — the image is never sent anywhere.

For video, it's reading a single snapshot of whatever frame is showing the instant you release the drag — pause the video first if you're after specific on-screen text, since a moving frame can come out blurry. DRM-protected video (most paid movies/shows) renders as black in screenshots by design, so OCR won't find anything there.

Keyboard shortcuts

No shortcuts are assigned by default — every action is always available from the toolbar menu. If you'd like a keyboard shortcut for text selection, image selection, or the color picker, open Settings from the popup and click "Set shortcuts," which takes you to Chrome's own shortcut editor at chrome://extensions/shortcuts.

Pick a Color (Page)

Choose Pick a Color (Page) from the popup (or press its shortcut if you've set one) and a small circle appears that follows your mouse, filled with a live preview of whatever pixel it's over. Move it around the page, then click your target to copy that color's hex code — the copy confirmation shows a swatch of what you picked. Press Esc any time to cancel without picking. It works by screenshotting the current tab and sampling pixels from that, so it can read anything rendered on the page, including video frames — but only within this browser tab, not other windows or apps on your screen.

Webpage Color Analyzer

Choose Webpage Color Analyzer from the popup to scan the whole page for colors actually in use (backgrounds, text, borders). A panel appears in the top-right listing swatches by how often each color shows up — click any swatch to copy its hex code. Click the panel's × or press Esc to close it.

Feedback

Choose Feedback from the popup to report a bug, request a feature, or send general feedback. It's sent directly to the developer, along with which page you were on and your browser info to help with debugging — an email is optional if you'd like a reply.

If nothing happens when you click

Extensions can't run on Chrome's own internal pages — chrome://extensions, a New Tab page, or the Chrome Web Store. If you try there, the toolbar icon shows a red ! badge for a few seconds. Switch to a normal website and try again.

Good to know

  • Text mode matches word by word, using each word's actual on-screen center point — so a word right on the edge of your drag box can go either way, similar to how native browser text selection behaves at line boundaries.
  • OCR reads whatever's visually in the image, so quality depends on resolution and clarity — small or blurry text may come out imperfect or blank, same as any OCR tool.
  • The color picker samples from a screenshot of the current tab, so it works consistently across browsers and operating systems — the trade-off is it can't reach outside the browser window to other apps or your desktop.
