Copy2Paste has a few actions, each of which can have its own keyboard shortcut. None are set by default — use the toolbar menu, or assign your own shortcuts below.

No shortcuts are assigned by default, so every action stays available from the toolbar menu with no keyboard conflicts. Chrome manages all extension shortcuts from one settings page for security reasons — the button above takes you straight there. Once it opens, find Copy2Paste in the list and click the pencil icon next to any action to record a key combination for it.
