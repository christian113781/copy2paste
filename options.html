Defaults are Alt+0 for text, Alt+Shift+0 for image, and Alt+Shift+C for the color picker. Chrome manages all extension shortcuts from one settings page for security reasons — the button above takes you straight there. Once it opens, find Copy2Paste in the list and click the pencil icon next to any shortcut to record a new key combination.
